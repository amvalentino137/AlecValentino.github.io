London Bike Riding Visualization

In this project, we download a Kaggle dataset containing data on different weather conditions for Londoners riding their bikes. We extract and transform the data in Python to clean the messy data and make it usable.
Finally, we load the data into Tableau and create different visuals to illustrate the number of bike rides in any given period, along with a customizable moving average and a heatmap to show temperature vs wind speed.
